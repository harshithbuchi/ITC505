Addendum:Description of the tags used

  • !DOCTYPE html: Says what kind of content and version of HTML is being used, in this case, HTML5.
  • h1 : Represents a top-level heading, used for the name of the person in this context.
  • p: Represents a paragraph of text.
  • div : Defines a division or section in the HTML document, often used as a container for grouping and styling purposes.
  • a : Creates a hyperlink with the specified URL. In this case, it links to the LinkedIn profile of the person.
  • section : Represents a section of the HTML document, often used to group related content together.
  • h2> : Represents a second-level heading, used for section headings in this context.
  • ul> : Defines an unordered list, typically used to list items without a specific order.
  • li : Represents a list item within an ordered or unordered list.
  • strong : Shows that something is very important or stands out, and is usually shown as bold text.
  • em : Usually shown as italicised text, this symbol stands for text that is emphasised.
  • ul and ol : Defines an unordered or ordered list, respectively.
  • br : Represents a line break, forcing text or elements following it to appear on a new line.
  • div class="portfolio-item" : Defines a division with a specific class, used to structure and style a portfolio item.
